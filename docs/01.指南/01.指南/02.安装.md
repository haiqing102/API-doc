::: warning 注意 ⚠️

- **JDK 8+**
- **Maven or Gradle**
  :::

## SpringBoot

### 🍊Maven

在项目的 pom.xml 的 dependencies 中加入以下内容:

```xml
<dependency>
    <groupId>vin.suki</groupId>
    <artifactId>api-sdk</artifactId>
    <version>1.0</version>
</dependency>
```

### 🍐Gradle

```
implementation group: 'vin.suki', name: 'api-sdk', version: '1.0'
```
